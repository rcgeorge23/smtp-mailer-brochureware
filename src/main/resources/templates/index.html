SMTP Trap

A simple and free fake SMTP server to use when automation testing your application.

No signup required.

Start using immediately.

1. Configure your app to use these SMTP details:

host={{smtpConfiguration.fakeSmtpHost}}
port={{smtpConfiguration.fakeSmtpPort}}
username=<anything>
password=<anything>
You are free to choose anything you like as a username and password. All requests will be accepted. The username and password you choose act as a composite key for your mailbox. You will need to use the same username and possword when retrieving your messages later.

2. Send an email

3. Get the message as JSON

http://{{smtpConfiguration.fakeSmtpHost}}/?username={{smtpMailContent.username}}&password={{smtpMailContent.password}}&toAddress={{smtpMailContent.toAddress}}
{{jsonMessage | json}}
